Using Safe Exam Browser (SEB)

Safe Exam Browser (SEB) is a secure web browser environment designed for online exams. It restricts access to unauthorized resources and ensures a safe testing environment. It also helpful to provide a controlled way to provide students access to tools like Desmos, Geogebra, or Wolfram Alpha in a secure environment.

For more information about SEB, visit the official SEB website.

What makes SEB safe?

  • Restricts access to other applications and websites during an exam.
  • Prevents copying, pasting, and screen capturing.
  • Allows configuration of permitted resources and tools.
  • Supports integration with various learning management systems (LMS).

SEB Compatibility

This website is optimized for use within Safe Exam Browser to ensure the best experience during exams, including verification of user agent, BEK, and CK. If any verification checks fail, a warning message will be displayed to the user. Please ensure you are using the latest version of SEB for optimal performance and security.

Suggested Configuration

  • Set the launch URL to DesmosMatrixGraphing.html to directly access the Desmos Matrix & CB Graphing tool within Safe Exam Browser.
  • The exam password should be written on the physical exam paper or provided securely by the instructor.
  • The quit password should be kept confidential and only shared with authorized personnel.
  • The quit password may also be posted on the exam room door or another secure location for students to access after testing.
  • The exam password and quit password should be changed every exam session to maintain security.
